# Application de gestion d’interventions de maintenance

## Objectif
Développer une application web permettant de créer, affecter, suivre et historiser des interventions techniques de maintenance.

## Contexte
Ce projet a pour but de répondre à un besoin opérationnel : centraliser la gestion des interventions, améliorer le suivi des techniciens et structurer l’historique des opérations.

## Fonctionnalités V1
- Authentification des utilisateurs
- Gestion des utilisateurs
- Gestion des équipements
- Gestion des interventions
- Affectation des techniciens
- Mise à jour des statuts
- Historique simple des opérations

## Rôles
- ADMIN
